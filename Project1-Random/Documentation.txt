I AM USING MY GRACE PERIOD

Jack Masson IGME 202 5

Description:
Used multiple forms of randomness to generate terrain. The project is an alien terrain where the player can destroy the incoming invasion.

User Responsibilty:
User will explore terrain and destroy aliens
C - Changes camera view
Q - Throws snowballs to destroy aliens

Above and Beyond:
Custom theme and added system to throw snowballs in the direction that the player is facing that destroy the aliens.
